获取作者列表

按照一定的顺序获取十条作者数据。

前端请求

  • 请求地址：
  • 请求方式：POST
  • 请求参数：
utydk: 1         // 检索类型 1：关注作者列表；2：我的关注列表；3：我的粉丝列表
geacg: 879646529 // 用户账号
iicje: 10        // 已获取的作者数

【说明】：utydk：检索类型。

  • 1-关注作者列表：根据用户账号，WHERE owner = 'geacg' 来检索当前用户关注的作者。
  • 2-我的关注列表：同关注作者列表，区别在于响应返回的内容。
  • 3-我的粉丝列表：根据用户账号，WHERE account = 'geacg' 来检索当前用户的粉丝。

获取作者排行榜

用户进入发现界面并点击作者排行榜按钮，即可进入到作者排行榜界面，此时应用会自动向后台发起请求以获取粉丝数最多的作者。

前端请求

  • 请求地址：
  • 请求方式：POST
  • 请求参数：
jvheh: 0        // 已获取的作者数
